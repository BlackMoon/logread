1. Ввести названия файлов, где содержатся шкалы и точки для расчета.
2. Ввести название получаемого *.dat - файла.
3. Формат *.txt - файла шкалы:

	название шкалы (произвольный набор символов)
X	Y	Z
/

/ - символ окончания

Порядок описания начинается с самой нижней шкалы.

4. Формат *.dat - файла точек:

NC	X	Y

5. Формат получаемого *.dat - файла:

NC	X	Y	Z	P	порода
